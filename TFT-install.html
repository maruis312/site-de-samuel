Configuration

PYTHON 3.7 REQUIRED

In order for the program to work on your machine please execute the file setup.py.

In your terminal enter :

python setup.py

It may take a few minutes

OR

Alternative method :

pip install -r requirements.txt

It may also take a few minutes

Execute

Once your machine is configured you can execute the file main.py via your terminal.

python main.py
